##  Project Overview
This project is a Python-based **Contact Book Management System** developed as part of Assignment 2.  
The objective is to implement a menu-driven application using:

- File Handling (CSV & JSON)  
- Exception Handling  
- CRUD Operations  
- Error Logging (Bonus Task)  

The program allows users to add, view, search, update, and delete contact information, and export or load data in JSON format.

---

##  Learning Objectives
- Understand **file I/O operations** using CSV & JSON  
- Implement **exception handling** in Python  
- Build a menu-driven program using loops and functions  
- Apply **CRUD operations** to manage data  
- Use **error logging** to record runtime issues  
- Format output neatly using f-strings  

---

##  Features Implemented

| Feature | Description |
|--------|-------------|
| **Add Contact** | Stores a new contact in CSV |
| **View Contacts** | Displays all saved contacts |
| **Search Contact** | Searches a contact by name |
| **Update Contact** | Modifies phone/email of a contact |
| **Delete Contact** | Removes a contact from CSV |
| **Export to JSON** | Converts CSV data to JSON file |
| **Load from JSON** | Reads and displays contacts from JSON |
| **Error Logging** | Logs exceptions into `error_log.txt` |
